# タスクボード: Python 数当てゲーム

`rich` ライブラリを使用したプレミアムな CLI ツールの開発進捗。

## 📅 開発フェーズ

### 1. 基盤構築 (Project Init)
- <Badge type="tip" text="DONE" /> **環境のセットアップ**
    - `venv` 仮想環境の作成
    - `rich` 依存関係のインストール

### 2. 設計 (Planning)
- <Badge type="tip" text="DONE" /> **実装計画書の作成**
    - ランクシステムや難易度設定の設計

### 3. コア開発 (Core Development)
- <Badge type="tip" text="DONE" /> **ゲームロジック**
    - 正解判定とヒント出力のアルゴリズム
    - 入力バリデーション（エラーハンドリング）
- <Badge type="tip" text="DONE" /> **CLI UI の磨き込み**
    - `rich` によるパネル・テーブル・色使いの最適化

### 4. 最終調整 (Final Polish)
- <Badge type="tip" text="DONE" /> **検証**
    - 複数難易度でのクリアテスト
    - 不正入力（文字入力など）への耐性確認
- <Badge type="tip" text="DONE" /> **ドキュメント化**
    - 練習ドリルの作成とコード埋め込み
